# torrent-s

A torrent searcher equipped with local socks5 proxy tunnel.

It's currently is not on hackage or stackage, so you need to build manually.

To build:
```shell
$ cabal sandbox init
$ cabal install
$ cabal build
$ ./dist/build/TorrentS/TorrentS "Catch me if you can"
```
